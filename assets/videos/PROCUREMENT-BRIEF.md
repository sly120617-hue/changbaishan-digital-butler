# 长白秘境网页正版视频采购简报

## 1. 使用场景

素材用于企业团建行程网页的全屏背景视频，公开互联网访问，无单独收费，不作为影视节目发行。所有素材必须允许：商业网站展示、剪辑、裁切、调色、静音循环播放和长期公开访问。

## 2. 采购底线

- 禁止购买或使用 Editorial Use Only（仅限编辑用途）素材。
- 授权主体应为实际使用公司或项目权利主体。
- 许可证不得限制网页访问人数或播放次数。
- 有清晰人物正脸时，必须具备肖像授权；无授权时应选择远景、背影或无人镜头。
- 不接受带水印、字幕、平台角标、音乐版权不明或嵌入式播放器素材。
- 防川边境素材不得包含敏感设施、军事区域或不适合公开展示的边防细节。

## 3. 技术规格

- 画幅：16:9 横屏。
- 首选分辨率：3840×2160；最低1920×1080。
- 编码：H.264 MP4，像素格式 yuv420p。
- 帧率：25/30fps，尽量统一。
- 单段成片：15—35秒。
- 网页交付码率：1080p约4—7Mbps；移动端可另做720p版本。
- 单文件目标：8—25MB。
- 无背景音乐，网页统一静音自动循环。
- 镜头运动平稳，避免急转、频闪、快速穿越和强烈变焦。

## 4. 逐场景验收标准

### 封面：长白山整体
必须含云海、群峰或天池揭幕镜头，首尾画面过渡自然。不能用普通雪山冒充长白山。

### D1：延吉夜景与朝鲜族民俗园
至少能识别延吉城市双语招牌、网红墙或中国朝鲜族民俗园传统建筑。可由2—3个短镜头拼接。

### D2：防川边境
优先龙虎阁、图们江、敬信湿地和三国交界大环境。要求画面表达“边境辽阔感”，不做边防设施特写。

### D3：长白山天池
必须明确识别火山口湖与长白山群峰。优先夏秋季、云雾流动、北坡公路或瀑布补充镜头。

### D4：森林漂流
优先森林河道、晨雾、皮筏远景与水面跟拍。人物应为环境比例，不以游客正脸为主体。

### 交通与清单章节
使用延边山地公路、林海、溪流、薄雾等低叙事强度镜头，避免抢夺正文注意力。

## 5. 交付资料

每段素材必须同时留存：

1. 原始下载文件；
2. 素材平台名称和素材编号；
3. 订单、发票或付款凭证；
4. 许可证PDF或授权页面存档；
5. 购买账号与购买日期；
6. 是否有人物肖像、建筑或景区限制的核对记录。

许可证与订单不上传到公开仓库，应存放在公司内部资料库；公开仓库只记录素材编号和授权状态。
