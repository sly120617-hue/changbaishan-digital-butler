## 技术验收标准

- 横屏 16:9
- 首选 3840×2160，最低 1920×1080
- H.264 MP4，建议 24/25/30fps
- 单段建议 15—30 秒，可自然循环
- 无字幕、无水印、无背景音乐
- 镜头运动平稳，避免快速旋转、频繁闪切和剧烈推拉
- 单文件建议压缩至 8—25MB，网页端总视频体积控制在合理范围
